# Contributing

Thank you for your interest in contributing to AI Test Case Generator!

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ai-test-case-generator.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`
5. Make your changes
6. Run tests: `npm test`
7. Type check: `npm run type-check`
8. Commit: `git commit -m "feat: your feature description"`
9. Push: `git push origin feature/your-feature-name`
10. Open a Pull Request

## Code Style

- Follow TypeScript best practices
- Use ESLint and Prettier (configured)
- Write tests for new features
- Update documentation as needed

## Commit Messages

Follow conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation changes
- `refactor:` Code refactoring
- `test:` Test additions/changes
